1. Type 'make' to compile the GTThreads library 
2. Type 'make matrix' to compile the matrix program
3. ./bin/matrix to run the matrix program
4. ./bin/matrix -s 0 to run with priority scheduler
5. ./bin/matrix -s 1 to run with credit scheduler
6. run with "-y" to enable yield function
7. run with "-lb" to enable load balancing
